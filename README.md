# To-Do List App

This project is a simple To-Do List web application.

The goal of the application is to allow users to create and manage tasks in a simple interface.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Dynamic task list

## Technologies

- HTML
- CSS
- JavaScript

## Purpose

This project was created to practice JavaScript and learn how to manipulate the DOM to build interactive web applications.

⚠️ This project is mainly a learning project.
